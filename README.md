# NSTC Law Journal Citation Style CSL

This repository provides the Citation Style Language (CSL) file for the *Law Journal Citation Style Example* (法學期刊引註格式凡例). This style was jointly established by the four Tier 1 TSSCI law journals in Taiwan.

本專案提供由**台灣四本 TSSCI 第一級法律學期刊共同研擬**之《法學期刊引註格式凡例》 Citation Style Language (CSL) 檔案，方便研究者於 Zotero 等文獻管理軟體中使用。

This project is an unofficial implementation,  and is not affiliated with the NSTC or the editorial boards of the aforementioned journals. No guarantee is given regarding the accuracy of the automatic citation format; users should exercise due diligence in verifying their citations.

本專案並非由國科會人文處、人社中心、法律學門或上述期刊編輯單位官方提供，並不擔保自動引用格式的完全正確性，使用者仍應謹慎檢查。

The CSL file in this project is adapted from the "法学引注手册（多语言）" CSL style developed by Zeping Lee and licensed under CC BY-SA 3.0. This project is released under the CC BY-SA 4.0 license.

本專案的 CSL 檔案，是基於 Zeping Lee 開發的「法学引注手册（多语言）」 CSL 檔案（採用 CC BY-SA 3.0 授權）修改而成，採用 CC BY-SA 4.0 授權。

Currently, this project only supports citation formatting for Chinese references. Citation formats for English, Japanese, German, and French references remain unchanged and follow the original 法学引注手册（多语言） style. It should be noted in particular that while the English version of 法学引注手册（多语言） largely follows The Bluebook, *The Example* is not entirely consistent with The Bluebook.

本專案目前僅支持中文文獻引註，英、日、德、法引註格式仍依「法学引注手册（多语言）」版本，尚未修改。尤其需要注意的是，該格式的英文版基本上參考 Bluebook，但《凡例》與 Bluebook 並非完全一致。

If you find any formatting errors or have suggestions for improvement, please open an issue or submit a pull request.

如果發現格式錯誤或有改進建議，歡迎提出 Issues 或提交 Pull Request。
